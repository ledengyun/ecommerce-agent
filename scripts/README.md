## ❓ 常见问题

### Q1: 显示"未找到商品元素"

**原因：**
- 页面还未完全加载
- 当前不是商品列表页
- 需要登录

**解决：**
1. 向下滚动页面，加载更多商品
2. 确认在搜索结果页（不是首页）
3. 刷新页面后重试
4. 登录 1688 账号

### Q2: 提取到的商品数量为 0

**解决：**
1. 检查页面是否是移动版（m.1688.com）
2. 尝试桌面版（www.1688.com）
3. 使用调试脚本查看详细原因

### Q3: 提取到的商品缺少价格

**原因：** 价格是动态加载的

**解决：**
1. 等待几秒让页面完全加载
2. 向下滚动触发加载
3. 手动刷新后重试

### Q4: JSON 格式不正确

**解决：**
1. 确保复制完整的 JSON（从 `[` 到 `]`）
2. 使用 JSON 验证工具检查格式
3. 或直接在 Web 控制台粘贴
