# 📥 1688 商品提取 - 自动保存版使用说明

## 🚀 快速开始（3 步完成）

### Step 1: 运行提取脚本

1. 打开 1688 搜索页（如搜索"跨境专供"）
2. 按 **F12** 打开开发者工具 → **Console**
3. 输入：`allow pasting` → Enter
4. 复制并粘贴脚本内容：
   ```bash
   cat /home/admin/.openclaw/workspace/ecommerce-agent/scripts/extract_1688_auto.js
   ```
5. 按 **Enter** 运行

### Step 2: 自动下载 JSON

脚本运行后会：
- ✅ 自动提取商品信息
- ✅ **自动下载 JSON 文件** 到浏览器的下载目录
- ✅ 文件名：`1688_products_时间戳.json`

### Step 3: 导入数据

运行导入脚本：
```bash
cd /home/admin/.openclaw/workspace/ecommerce-agent/scripts
chmod +x import_downloaded.sh
./import_downloaded.sh
```

脚本会自动：
- 🔍 查找最近下载的文件
- 📋 复制到 `data/1688_products.json`
- 🗄️ 询问是否导入数据库
- ✅ 完成！

## 📁 脚本说明

### extract_1688_auto.js

**功能：**
- ✅ 智能提取商品信息（标题、价格、图片、链接）
- ✅ 自动下载 JSON 文件（无需手动复制）
- ✅ 同时输出到 Console（方便查看和备份）

**下载的文件：**
```
文件名：1688_products_2026-03-12T14-45-00-000Z.json
位置：~/Downloads/
格式：JSON
```

### import_downloaded.sh

**功能：**
- 🔍 自动查找最近下载的 1688 文件
- 📋 复制到项目数据目录
- 🗄️ 可选：导入 MySQL 数据库
- 📊 显示商品统计

**用法：**
```bash
# 自动模式（查找最近的文件）
./import_downloaded.sh

# 手动指定文件
./import_downloaded.sh /path/to/your/1688_products.json
```

## 🐛 常见问题

### Q1: 没有自动下载文件？

**检查：**
1. 浏览器是否阻止了下载
2. Console 是否有错误信息
3. 是否提取到商品（看日志）

**解决：**
- 允许浏览器下载
- 刷新页面后重试
- 手动复制 Console 输出的 JSON

### Q2: 导入脚本找不到文件？

**解决：**
```bash
# 手动指定文件路径
./import_downloaded.sh ~/Downloads/1688_products_xxx.json

# 或手动复制
cp ~/Downloads/1688_products_xxx.json ../data/1688_products.json
```

### Q3: 提取到的商品数量为 0？

**解决：**
1. 确认在搜索结果页（不是首页）
2. 向下滚动加载更多商品
3. 刷新页面后重试
4. 登录 1688 账号
